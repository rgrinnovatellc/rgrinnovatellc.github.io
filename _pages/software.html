Earthquakes by RGR

Play Store

Android app for real-time earthquake data. Shows nearby events on a map, supports location search, and filters by distance and magnitude.

Problem

Earthquake data is hard to filter to what's actually nearby.

Platform

Android. Google Maps, Google Places, USGS GeoJSON feed.

Engineering focus

API integration, map rendering, real-time data handling.

SmartConvert

Offline Android toolkit: image conversion with document scanning, unit conversion, color conversion, a screen ruler, and optional ARCore room measurement. No ads, no accounts, no network needed.

Problem

These tasks exist across five different apps, most with ads.

Platform

Android. Offline core; optional ARCore. Exports to JPEG, PNG, WebP, BMP, TIFF, PDF.

Engineering focus

Document edge detection, perspective correction, on-device processing, no data collection.

Capabilities

Features

  • Fetches real-time earthquake data from the USGS GeoJSON API.
  • Displays event locations through a map-first mobile experience.
  • Supports address suggestions and radius-based filtering.
  • Provides image conversion with document edge detection and six export formats including PDF.
  • Includes unit conversion, color conversion, a calibrated screen ruler, and optional ARCore room measurement.
