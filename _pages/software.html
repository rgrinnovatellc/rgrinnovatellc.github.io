Software

Software published through RGR Innovate LLC. Source code on GitHub.

Android applications

Earthquakes by RGR

Displays real-time earthquake data from the USGS GeoJSON API on a map. Supports location search, radius-based filtering, and magnitude filtering. Uses Google Maps and Google Places.

Google Play

SmartConvert

Offline Android toolkit for image conversion (with document scanning and edge detection), unit conversion, color conversion, screen ruler, and optional ARCore room measurement. Exports to JPEG, PNG, WebP, BMP, TIFF, and PDF. No ads, no accounts, no network required.

In development.
